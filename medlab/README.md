# 🧬 MedLab — PWA для медицинских анализов

Прогрессивное веб-приложение для хранения, отслеживания и анализа медицинских показателей.

## 🚀 Быстрый запуск

### Требования
- Node.js 16+

### Установка и запуск

```bash
# Распакуйте архив
cd medlab

# Установите зависимости
npm install

# Запустите сервер
npm start
```

Откройте в браузере: **http://localhost:3000**

## ✨ Возможности

- **Аутентификация** — регистрация и вход с JWT-токенами
- **Анализы** — добавление результатов с параметрами, значениями, единицами измерения и референсными диапазонами
- **Категории** — кровь, моча, биохимия, гормоны, витамины и др.
- **Статусы** — автоматическое определение отклонений от нормы
- **Графики** — динамика показателей во времени (Chart.js)
- **PWA** — работает как нативное приложение, кешируется для офлайн-работы
- **Экспорт** — выгрузка всех данных в JSON

## 🏗 Структура

```
medlab/
├── server.js          # Express API + аутентификация
├── public/
│   ├── index.html     # SPA frontend
│   ├── css/app.css    # Стили
│   ├── js/app.js      # Логика приложения
│   ├── manifest.json  # PWA manifest
│   └── sw.js          # Service Worker
└── data/              # JSON-файлы (создаются автоматически)
    ├── users.json
    └── tests.json
```

## 🔐 Безопасность

- Пароли хешируются через **bcrypt** (12 раундов)
- Сессии через **JWT** (30 дней)
- Данные каждого пользователя изолированы

## 🌐 Деплой

Для продакшена установите переменную окружения:
```bash
JWT_SECRET=your-super-secret-key PORT=3000 npm start
```

## 📱 Установка как PWA

В браузере (Chrome/Safari) нажмите «Установить приложение» или «Добавить на главный экран».
